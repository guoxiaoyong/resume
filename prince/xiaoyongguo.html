Personal Projects

I develop small software in my spare time simply for fun. The following are a few of them:

  • visual pattern communications - Using computer screen as the signal transmitter and camera as the signal receiver for data transmission. Binary data is encoded into a sequence of QR code which is displayed on the screen frame by frame and captured by camera on another computer. Currently the data transmission rate is slightly more than 4KB/s.
  • data communication over voice channel - Using frequency-shift-keying modulation scheme to transform binary data into voice signal. Voice signal is fed into soundcard and transmitted via audio cable or over the air via speaker (use microphone as receiver).
  • ripple effect simulation - display visual effection of ripples on any picture by solving the 2D wave equation.
  • gxyos - A very simple operating system developed for better understanding of x86 architecture and internals of OS kernel.
